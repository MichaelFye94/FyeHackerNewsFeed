# FyeHackerNewsFeed

## Requirements
Built with:
* .NET Core 3.1
* Angular 8.2+
* Visual Studio 2019

## Build
Build tied to debug IIS Express run in vs
This will start both .NET Service and Angular client

## Tests
Angular tests must be run from hacker-news-feed/ClientApp directory
